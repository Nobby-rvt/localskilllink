LocalSkillLink

A hyperlocal skill exchange platform where users trade skills instead of money.

Problem
Many people have skills but no income source. Others need help but cannot afford services.

Solution
Users earn credits by helping others and spend credits to get help.

Features
- Skill profiles
- Skill requests
- Credit system
- Location filtering
- Ratings
- Chat system

Tech Stack
Frontend: HTML, CSS, JavaScript  
Backend: Node.js, Express  
Database: MongoDB

Installation

1. Clone the repository
2. Run npm install
3. Start server with: node server.js
